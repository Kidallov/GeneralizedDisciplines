# Инопланетное вторжение

Практическая работа 2: развитие игры на `pygame`

## Цель работы

Доработать базовую игру "Инопланетное вторжение": добавить звуковые эффекты, систему уровней с ростом сложности, сохранение и загрузку прогресса, интерактивные игровые бонусы и скрипты сборки исполняемого файла.

## Структура проекта

- `alien_invasion.py` - главный файл игры: запуск, игровой цикл, обработка событий, столкновения, бонусы, сохранение/загрузка, переходы между уровнями.
- `settings.py` - статические и динамические настройки игры: размеры окна, скорости, лимиты, параметры бонусов и пути к файлам сохранения.
- `ship.py` - класс корабля игрока и логика движения вправо/влево.
- `bullet.py` - класс снаряда; обычные и усиленные снаряды отличаются шириной, цветом и количеством пробитий.
- `alien.py` - класс инопланетянина и проверка выхода флота за края экрана.
- `bonus.py` - интерактивные объекты: дополнительная жизнь, щит и усиленные снаряды.
- `game_stats.py` - текущая статистика, жизни, счет, уровень, активные эффекты, сохранение прогресса через `pickle` и сохранение рекорда.
- `scoreboard.py` - отображение счета, рекорда, уровня и количества кораблей.
- `button.py` - кнопка запуска игры.
- `sound_effects.py` - процедурные звуковые эффекты выстрела, попадания, потери жизни, бонуса, щита и завершения игры.
- `build_windows.bat`, `build_macos.sh`, `build_linux.sh` - скрипты сборки через PyInstaller.
- `resources/` - папка ресурсов, подключенная к скриптам сборки.
- `requirements.txt` - зависимости `pygame` и `pyinstaller`.

## Запуск

```bash
python3 -m pip install -r requirements.txt
python3 alien_invasion.py
```

## Управление

- Стрелки влево/вправо - движение корабля.
- Пробел - выстрел.
- `P` или `Enter` - начать новую игру.
- `S` - сохранить текущий прогресс в `savefile.pkl`.
- `L` - загрузить прогресс из `savefile.pkl`.
- `Q` - выход.

## Реализованные задания

### 1. Звуковые эффекты

В игре используется класс `SoundEffects`, который создает звуки средствами `pygame.mixer.Sound`. Добавлены события:

- выстрел корабля;
- уничтожение инопланетянина;
- потеря жизни;
- подбор бонуса;
- срабатывание щита;
- завершение игры.

Если аудиоустройство недоступно, класс безопасно отключает звук, не прерывая игровой процесс.

### 2. Система уровней

Уровень хранится в `GameStats.level`. Когда весь флот уничтожен:

- снаряды очищаются;
- уровень увеличивается на 1;
- флот создается заново;
- скорости корабля, снарядов и пришельцев увеличиваются через `Settings.increase_speed()`;
- очки за уничтожение пришельца растут через множитель `score_scale`.

### 3. Сохранение и загрузка прогресса

Сохранение реализовано через стандартный модуль `pickle`.

В файл `savefile.pkl` записываются:

- текущий счет;
- рекорд;
- уровень;
- оставшиеся жизни;
- оставшееся время действия щита;
- оставшееся время действия усиленных снарядов.

При загрузке игра восстанавливает уровень, счет, жизни и активные эффекты. Скорость врагов пересчитывается по номеру уровня, затем создается новый флот.

### 4. Интерактивные объекты

Добавлен класс `Bonus`. Бонусы с заданной вероятностью появляются на месте уничтоженного инопланетянина и падают вниз. Столкновение с кораблем определяется через `pygame.sprite.spritecollide`.

Типы бонусов:

- `life` - восстанавливает одну жизнь, но не больше стартового лимита;
- `shield` - временно защищает корабль от столкновения или достижения пришельцами нижней границы;
- `power` - временно включает усиленные снаряды, которые могут уничтожить несколько пришельцев одним выстрелом.

Активные эффекты отображаются на экране с таймером. Щит дополнительно рисуется вокруг корабля.

### 5. Сборка исполняемого файла

Для сборки добавлены скрипты:

```bash
build_windows.bat
./build_macos.sh
./build_linux.sh
```

Скрипты используют PyInstaller:

```bash
pyinstaller --onefile --add-data "resources:resources" alien_invasion.py
```

Для Windows используется разделитель `;`, для macOS и Linux - `:`. Сборку необходимо выполнять на той операционной системе, для которой создается исполняемый файл.
