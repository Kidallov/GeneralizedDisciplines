# Краткий отчет работы приложения

### **Имеется 3 приложения `itproger`, `main`, `news` - именно так называются такие папки в django.**

### Файл `manage.py` является главным - именно он запускает наш сервер.

## Основные команды, которыми я пользовался при создании мини-проекта:

  * Чтобы запустить сервер `python3 manage.py runserver`

  * Создать админа `python3 manage.py createsuperuser`

  * Создать новое приложение `python3 manage.py startapp name_app`

  * Выполнить миграцию `python3 manage.py migrate`
