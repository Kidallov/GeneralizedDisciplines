# Лабораторная работа № 2

**Тема:** Сравнение теоретических и эмпирических методов исследования

**Цель:** провести сравнительный анализ теоретических и эмпирических методов исследования, применить их на практике для решения конкретной ИТ-задачи, проанализировать преимущества и ограничения каждого из используемых методов

**Задачи:**

1. Изучить сущность, виды и особенности теоретических и эмпирических методов исследования.
2. Выбрать ИТ‑задачу и решить её, используя сначала теоретические, затем эмпирические методы.
3. Сравнить результаты, точность, временные затраты и применимость методов.
4. Сформулировать рекомендации по выбору метода для аналогичных задач.

---

### Этап 2. Практическая часть. Решение ИТ-задачи теоретическим и эмпирическим методом

1. **Выберите** одну из предложенных задач или предложите свою (согласовав с преподавателем):
   - оптимизация алгоритма сортировки для больших массивов данных;
   - анализ производительности веб‑приложения при разной нагрузке;
   - сравнение эффективности двух методов сжатия данных;
   - оценка времени отклика мобильного приложения при разных сетевых условиях.

2. **Обоснуйте** выбор теоретических методов. Используя обоснованно выбранные теоретические методы, проведите анализ/моделирование/формализацию и т.д.

3. **Обоснуйте** выбор эмпирических методов. Проведите эксперимент:
   - измерьте необходимые показатели (время выполнения, количество ошибок и т.д.);
   - визуализируйте полученные результаты (постройте графики/таблицы/диаграммы).

---

### Этап 3. Сравнение и анализ

Сравните результаты применения двух методов по критериям:
- точность прогноза (насколько теоретические оценки совпали с практикой);
- временные затраты на исследование;
- ресурсоёмкость (оборудование, ПО, данные);
- применимость для других задач;
- ограничения метода.
