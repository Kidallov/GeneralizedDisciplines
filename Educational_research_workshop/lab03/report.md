# Лабораторная работа №3 "Разработка плана эксперимента по проверке корректности и эффективности проектных решений"

## Тема исследования
Исследование метода сжатия изображений без потерь с использованием библиотеки PIL.

## Описание проектного решения

Рассматривается программное решение для автоматического сжатия изображений форматов JPG и PNG.

### Проблема
Хранение и передача изображений требует значительных ресурсов (диск, сеть). Необходимо уменьшить размер файлов без потери качества.

### Решение
Реализован скрипт на Python, который:
- проходит по всем изображениям в папке `images/`
- открывает изображения с помощью библиотеки PIL
- сохраняет их в формате PNG с максимальным уровнем сжатия (`compress_level=9`)
- использует оптимизацию (`optimize=True`)

### Особенности реализации
- пакетная обработка файлов
- отсутствие потерь качества (lossless compression)
- автоматическое именование выходных файлов (`compressed_*`)

### Ожидаемые преимущества
- уменьшение размера файлов
- сохранение качества изображений
- простота и универсальность решения

---

## Критерии оценки корректности и эффективности

### Корректность
- успешная обработка всех файлов без ошибок
- корректное открытие и сохранение изображений
- сохранение визуального качества изображения

### Эффективность
- степень сжатия (уменьшение размера файла, %)
- время обработки изображений (мс)
- использование памяти (опционально)

---

## План эксперимента

### 1. Цель
Оценить эффективность метода сжатия изображений без потерь.

### 2. Гипотеза
Использование параметров `optimize=True` и `compress_level=9` позволяет уменьшить размер изображений без потери качества.

### 3. Методы и методология

Методы:
- пакетная обработка изображений
- сравнение размеров файлов до и после сжатия
- визуальная проверка качества

Инструменты:
- Python
- библиотека PIL (Pillow)
- стандартные средства ОС для измерения размера файлов

Метрики:
- размер файла (КБ/МБ)
- коэффициент сжатия (%)
- время обработки (мс)

### 4. Условия проведения
- папка с изображениями (`images/`)
- форматы: JPG, PNG
- одинаковые условия выполнения (одна система)

### 5. Этапы проведения
1. Подготовка набора изображений
2. Запуск программы сжатия
3. Сбор данных (размер до/после)
4. Анализ результатов

### 6. Анализ результатов
Сравнение исходных и сжатых изображений:
- вычисление процента уменьшения размера
- оценка визуального качества

## Результаты пилотного эксперимента

Пример результатов:

| Файл | Размер до (КБ) | Размер после (КБ) | Сжатие (%) |
|------|---------------|------------------|------------|
| image1.jpg | 1200 | 850 | 29% |
| image2.png | 980 | 700 | 28% |

Среднее уменьшение размера: ~25–30%

Визуальных потерь качества не обнаружено.

## Выводы

В ходе эксперимента было установлено:

- метод сжатия без потерь позволяет существенно уменьшить размер изображений;
- использование максимального уровня сжатия эффективно для PNG;
- визуальное качество изображений сохраняется;
- метод подходит для задач хранения и передачи данных.

Ограничения:
- увеличение времени обработки при высоком уровне сжатия;
- не всегда эффективно для уже сжатых PNG.
