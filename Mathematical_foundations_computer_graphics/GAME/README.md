# Инопланетное вторжение

Практическая работа 1: базовая 2D-игра на `pygame`.

## Запуск

```bash
python3 -m pip install -r requirements.txt
python3 alien_invasion.py
```

В этой папке уже создано локальное окружение `.venv` с установленным `pygame`, поэтому можно запустить сразу так:

```bash
.venv/bin/python alien_invasion.py
```

## Управление

- Стрелки влево/вправо - движение корабля.
- Пробел - выстрел.
- `P` или `Enter` - начать новую игру.
- `Q` - выход.

## Что реализовано

- Окно игры через `pygame.display.set_mode`.
- Главный цикл с обработкой событий, обновлением объектов и перерисовкой экрана.
- Классы `Ship`, `Bullet`, `Alien`.
- Флот пришельцев, который движется горизонтально, меняет направление у края экрана и опускается вниз.
- Столкновения пуль с пришельцами, столкновения корабля с пришельцами и проверка нижней границы.
- Система жизней, уровней, счета и растущей сложности.
- Таблица рекордов с сохранением лучшего результата в `high_score.json`.
- Процедурные звуковые эффекты с безопасным отключением, если аудиоустройство недоступно.
